# 📈 Live Stock Market Dashboard

An interactive stock market dashboard built with Python.

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
- yfinance (live data)

## ✨ Features
- Live stock prices from Yahoo Finance
- Interactive charts (Bar & Line)
- Compare two companies side by side
- Auto refresh every 5 minutes
- Download data as CSV

## 🚀 How to Run
pip install streamlit plotly yfinance pandas
streamlit run dashboard.py Plotly
